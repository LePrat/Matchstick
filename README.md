# Matchstick

To launch the game :

```sh
$ make
$ ./Matchstick $Line_amount $Match_limit
```
Example :

```sh
$ ./Matchstick 5 7
```

How to play :
The game asks you to chose a line, then the amount of stick you want to remove starting from the right of the line.
The goal is to make the AI remove the last match.
Good luck !!
